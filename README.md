# Flow_organization_database

This repo is the raw data used in the manuscript "Flow organization in mixed convection at shear-buoyancy equilibrium state for heavy liquid metals".

Operating condition "Re180Ri0" means $Re_{\tau,N}=180, Ri=0$, "Re180Ri1" means $Re_{\tau,N}=180, Ri=1$.

2026.6.15 update:

The manuscript is renamed as "Flow organization in unstably stratified mixed convection at $Ri=1$ for heavy liquid metals".

The LCS-u and LCS-theta files are updated with the sensitivity check of domain size, grid resolution, and filter parameters.
